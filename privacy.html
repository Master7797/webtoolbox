Privacy Policy

Effective: May 23, 2026

Data Collection

Web Toolbox does NOT collect, store, or transmit any personal data.

All tools in this application are executed entirely client-side in your browser using JavaScript. No information you enter into any tool — whether text, files, or configuration — is sent to any server, API, or third party.

Cookies

This site does not use cookies, tracking scripts, or fingerprinting technologies.

Third-Party Services

  • Hosting: Static files are served via Surge.sh CDN (a content delivery network). The CDN may log standard HTTP request metadata (IP address, timestamp, requested page) for operational purposes. This data is not accessible to us and is handled per Surge.sh's privacy policy.
  • Donations: The Ko-fi donation link opens an external site. Ko-fi's privacy policy applies once you leave our site.

Data Security

Since no user data is collected or stored, there are no data security risks associated with using this tool. Your processing stays in your browser tab.

Changes to This Policy

If this policy changes in the future, the updated date at the top will reflect the change. Significant changes will be noted on the site.

Contact

For questions about this privacy policy, reach out via Ko-fi.
